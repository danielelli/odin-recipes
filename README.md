# odin-recipes

![image](https://github.com/danielelli/odin-recipes/assets/90986300/ab01e754-b5c9-4954-a39e-873d45eb92c8)

**A minimalistic recipe website that is simple yet sweet (or savory, depending on the recipe 😄)**

To locally install this project, clone the repository:

`git clone git@github.com:danielelli/odin-recipes.git`

To run, a good recommendation is [live-server](https://www.npmjs.com/package/live-server):

`npm install -g live-server`

or use the [Live Server](https://github.com/ritwickdey/vscode-live-server) VSCode extension.
